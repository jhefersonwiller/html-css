Sobre a Tag "a href"

a
Âncora.
href
hypertext reference.
hreflang="en"
Usado para indicar qual é o idioma principal para onde o link está direcionando.

Parâmetros da Tag

target
Link alvo
rel
Relativo a
  • Opções do parâmetro "rel"
    • external: Para links externos de terceiros.
    • next: Link para uma próxima página do seu site.
    • prev: Link para uma página anterior do seu site.
    • nofollow: Link para página de terceiros onde o SEO não deve dar importância.
    • author:: Link para página do autor do artigo atual.

  • Opções do parâmetro "target"
    • _blank: Abre o link em uma nova aba.
    • _self: Abre o link na aba atual (usado quando a naveção deve ser mantida no site).
